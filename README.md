# WBS-User-Panel (Customer-Facing Menu)

Welcome to the **WBS-User-Panel**, the interactive digital menu for the WBS Menu Demo. This application provides a seamless, mobile-optimized experience for customers to browse the menu and place orders.

## Key Functionality

- **Digital Menu Browsing:** Browse categories, view item details, and add products to the cart.
- **Responsive Design:** A mobile-first, app-like experience with smooth animations.
- **Shopping Cart & Checkout:** Easy-to-use cart management and checkout flow.
- **Form Validations:** Secure and validated inputs for customer details.
- **Dynamic Interactions:** Interactive elements like carousels for featured items.

## Technology Stack & Tools

### Core

- **Framework:** [Next.js 14](https://nextjs.org/)
- **Language:** JavaScript / TypeScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Package Manager:** NPM

### Database & Auth

- **Database:** Supabase
- **Backend Services:** Supabase (`@supabase/supabase-js`)

### UI & UX

- **Component Primitives:** Radix UI (`@radix-ui/react-*`) & Headless UI (`@headlessui/react`)
- **Animations:** Framer Motion & `tailwindcss-animate`
- **Icons:** Lucide React
- **Forms:** React Hook Form (`react-hook-form`), Zod (`zod`), and Hookform Resolvers
- **Input Masking:** React Input Mask (`@react-input/mask`) for phone numbers and specific inputs.
- **Carousels:** Embla Carousel React (`embla-carousel-react`)
- **Notifications:** Sonner (`sonner`) for toasts.
- **Dates:** `date-fns` and `react-day-picker`
